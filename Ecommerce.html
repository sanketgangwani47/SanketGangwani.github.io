E-COMMERCE SALES DASHBOARD

PROBLEM

The project was made for an E-commerce store operating in various states of United States. The store wants to know their business performance in last few years and has asked to prepare a report displaying various KPI's and charts through which it can make better decisions.

SOLUTION

To help the client meet their requirements, I've created an Excel Dashboard that displays number of necessary KPI's and visualizations. To ensure trueness of data, I've cleaned the data using basic excel functions and later by using pivot tables, I've created several charts that provide following information:

  • The trend analysis shows that sales and profit are highest in November and December.
  • Technology sector contributes the most in total sales (approximately 36%).
  • Demographic analysis shows that California state has provided maximum sales by contributing an approximate 19% in total sales.
