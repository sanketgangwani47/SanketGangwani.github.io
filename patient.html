Patients emergency visits report

PROBLEM

The purpose of this project was to analyze a hospital’s emergency visits report and obtain some insights including patient’s statistics, Trends, Gender biases, Race biases, etc.

SOLUTION

To meet the project objective, I've created a Microsoft Power BI Dashboard that displays number of necessary KPI's and visualizations. Utilizing Power Query and DAX, I've incorporated additional columns and measures in order to understand data effectively.

The dashboard insights suggests that the hospital has seen approximately 9216 patients where almost 50% patients are appointed via administrative appointment. Several trend analyses depict the statistics of patient arrival where we can see that patient count has been increased in 2020 as compared to 2019 where August month has observed maximum patients. It is also relevant that Adult age-group contributes highest approximately 77% in total patient count. Utilizing heatmaps we observe that there is no definite pattern in average waiting time/average satisfaction for any race/age-group, by which we can conclude that there are no Race biases.
